# LingLuoHome 功能说明指南

个人主页使用 Astro 构建，极简风格，支持多种内容展示。

---

## 📂 核心配置文件

| 文件路径 | 用途 |
|---------|------|
| `src/data/site.ts` | 站点基本信息（标题、头像、社交账号、统计配置） |
| `src/data/projects.ts` | 项目展示数据 |
| `src/data/friends.ts` | 友链数据 |
| `src/data/photos.ts` | 照片墙数据 |

---

## 🏠 首页 ([src/pages/index.astro](src/pages/index.astro))

自动统计并展示：
- 最新文章列表
- 项目卡片
- 照片墙预览
- 链接快捷入口

## 📝 发帖 / 文章 ([src/pages/notes.astro](src/pages/notes.astro))

### 添加新文章

在 `src/content/posts/` 目录下创建 `.md` 文件：

```markdown
---
title: "文章标题"
date: 2026-06-14
description: "文章描述，用于搜索和列表展示"
tags: ["标签1", "标签2"]
category: "分类"
---

## 正文内容

这里是你的文章内容，支持 Markdown 语法。
```

### 支持功能
- Markdown 渲染
- 代码高亮
- 数学公式（KaTeX）
- 文章内目录导航
- 全局搜索（Ctrl+K）

## 🎵 音乐 ([src/pages/music.astro](src/pages/music.astro))

基于 React + 网易云 API 的音乐播放器，支持歌词同步。

### 配置歌曲列表

编辑 [src/components/music/config.ts](src/components/music/config.ts)：

```typescript
export const songIds = ["歌曲ID1", "歌曲ID2", "歌曲ID3"];
```

### 获取歌曲 ID

从网易云音乐歌曲 URL 中提取：
- 歌曲链接：`https://music.163.com/#/song?id=298330`
- ID = `298330`

### 功能特性

- 歌曲信息自动获取（标题、歌手、封面）
- 歌词同步显示（打字机效果）
- 播放控制（播放/暂停、切歌、进度拖拽）
- 播放模式（循环、单曲、随机）
- 浮动播放器（右下角）

### API 说明

后端 API：`/api/music` 代理网易云 API 请求，自动获取歌曲详情和歌词。
