# LittleHome

极简风格的个人主页，使用 [Astro](https://astro.build) 构建。

## 功能

- **待办清单** — 添加、勾选、删除任务，数据存储在本地
- **快速笔记** — 即时记录想法，支持 Markdown 文章发布
- **文章系统** — 基于内容集合的静态博客，自动生成文章列表

## 本地开发

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:4321`

## 构建

```bash
npm run build
```

静态文件输出到 `dist/` 目录。

## 技术栈

- [Astro](https://astro.build) — 静态站点生成
- TypeScript
- CSS 自定义属性
- localStorage 本地持久化
